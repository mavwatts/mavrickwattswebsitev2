Marvel Comics Landing Page

Marvel Comics is the brand name and imprint offering access to over 25,000 digital comics including fan favorite characters like Spiderman, Captain America, Ironman, Thor, and more.

OVERVIEW


My goal for this project was to create a simple yet straightforward digital comic book service site for users to navigate. I needed to make it easy for users to understand the comic service packages and benefits they get from signing up for the service. Aside from design, responsiveness and utilizing javascript were top priority in this project.



CHALLENGES


The only challenge that I had with this project was the tab switch reveal. I wanted to make sure site visitors knew exactly which tab they were under so I incorporated a little bit of javascript to make a red bar underneath the selected icon and css to make sure the icon was brighter than the rest.




THE SOLUTION


The solution I came up with for creating this landing site was to ensure users visiting understand the focus is to get them to sign up for the service. As soon as you land on the site a call to action (COA) is made visible in the form of a free 30 day trial. Below the hero section I wanted to give the benefits to signing up via different tabs the user can interact with. I did this by utilizing Javascript and CSS to let users know where they should click next to find out more about the services offered. I used Netflix's landing page for inspiration and incorporated it into this project to give it a modern feel with comic book characters as the main focus. The solution I came up with to display the pricing data was to use a table and a simple tab switch to show the offers.
